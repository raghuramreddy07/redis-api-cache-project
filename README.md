# 🚀 Redis API Cache Project

A full-stack application demonstrating Redis API caching with Node.js, Express.js, MongoDB, and React.

## Features

- Product CRUD APIs
- Redis API Response Caching
- Cache Hit / Cache Miss
- Cache-Aside Pattern
- Cache Invalidation
- TTL (Time-To-Live)
- React Frontend
- REST APIs

## Tech Stack

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- Redis

### Frontend
- React
- Vite
- Tailwind CSS
- Axios

## Project Structure

```
backend/
frontend/
```

## Getting Started

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
